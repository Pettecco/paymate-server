# PayMate Server

Backend for **PayMate**, a subscription and payment management system.  
Allows users to manage individual and group subscriptions, track payments, and send reminders.

## Features

- Manage users, subscriptions, and payments
- Support for group subscriptions
- Generate and run migrations with TypeORM
- Dockerized PostgreSQL database
- Environment variable configuration

## Requirements

- Node.js >= 18
- PostgreSQL
- Docker (optional, for running PostgreSQL in a container)
